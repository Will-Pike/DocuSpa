Step 2: Authorize DocuSpa

After clicking the authorization URL above, you'll be redirected to ShareFile to login and authorize DocuSpa.

ShareFile will then redirect you back with some parameters in the URL. Look for these parameters:

  • code - Authorization code
  • subdomain - Your ShareFile subdomain
  • apicp - API control plane (e.g., sharefile.com)
  • appcp - App control plane
